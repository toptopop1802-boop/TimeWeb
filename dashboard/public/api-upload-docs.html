Image Upload API

Быстрая и надежная загрузка изображений

Основная информация

Метод
POST
Endpoint
/api/images/upload
Авторизация
Bearer Token
Поле формы
image

Требования к файлам

Макс. размер
15 MB
Поддерживаемые форматы
PNG JPG JPEG GIF WebP

Пример запроса

cURL 
curl -X POST https://bublickrust.ru/api/images/upload \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "image=@your-image.png"
