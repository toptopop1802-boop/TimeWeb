===================================================
ИНСТРУКЦИЯ ПО РАЗВЕРТЫВАНИЮ на СЕРВЕР
===================================================

Файл .txt содержит ВСЕ секреты!
Этот файл ТОЛЬКО для ручной загрузки на сервер!

===================================================
ШАГ 1: ЗАГРУЗКА НА СЕРВЕР
===================================================

Через WinSCP / FileZilla / SFTP:

ОТКУДА: W:\Projects\dsbot\dashboard\.txt
КУДА: /root/TimeWeb/dashboard/.txt


===================================================
ШАГ 2: НА СЕРВЕРЕ
===================================================

Выполните команды:

cd /root/TimeWeb/dashboard

# Удалить старый .env
rm -f .env

# Скопировать .txt в .env
cp .txt .env

# Установить правильные права
chmod 600 .env

# Проверить содержимое
cat .env

# УДАЛИТЬ .txt после копирования!
rm -f .txt


===================================================
ШАГ 3: ПРОВЕРКА
===================================================

Убедитесь что в .env есть:
- SUPABASE_KEY (длинный ключ)
- DISCORD_BOT_TOKEN (начинается с MTQx)
- DISCORD_CLIENT_SECRET (PkdDkT3v...)
- DISCORD_CLIENT_ID (1417959083704582224)


===================================================
ШАГ 4: ПЕРЕЗАПУСК СЕРВЕРА
===================================================

pm2 restart dashboard

ИЛИ

npm start
